Sobre o Desenvolvedor

Este site foi criado como parte de um trabalho da faculdade por Vinicius Sousa Lucena Chinaqui, estudante do Instuto Federal de São Paulo (IFSP). Atualmente, estou no segundo semestre do curso e estou desenvolvendo minhas habilidades em desenvolvimento web com foco em HTML, CSS e JavaScript.

Missão

Proteger os jogos online, garantindo um ambiente justo para todos os jogadores.

Motivação para o Projeto

Este projeto foi desenvolvido para aplicar os conceitos de desenvolvimento web aprendidos no curso, com foco em criar um site funcional. A escolha do tema "Anti-Cheats" foi feita devido ao interesse em explorar como os sistemas de segurança em jogos funcionam e ajudam a garantir um ambiente justo para todos os jogadores.
